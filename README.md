# Send Content to Web

Ứng dụng Android đơn giản cho phép:
- Nhập hoặc dán nội dung văn bản
- Nhấn nút gửi
- Nội dung sẽ được gửi lên website thông qua API (PHP)

## Chức năng
- Gửi nội dung từ Android lên Web
- Kết nối bằng HTTP POST
- Cấu trúc đơn giản, dễ mở rộng

## Cấu trúc thư mục
- android-app/: mã nguồn Android
- web/: mã nguồn phía server (PHP)

## Công nghệ sử dụng
- Android (Java)
- PHP
- HTTP API
